# 💰 ماشین حساب حقوق (Overtime Pay Calculator) - Python

---

## ✨ توضیح پروژه

این پروژه یک برنامه ساده به زبان **Python** است که میزان حقوق یک کارمند را بر اساس **ساعات کاری** و **نرخ ساعتی** محاسبه می‌کند.  
در این برنامه، **اضافه‌کاری (Overtime)** نیز در نظر گرفته شده است.

---

## 🚀 ویژگی‌ها

- ⏰ دریافت ساعات کاری از کاربر
- 💵 دریافت نرخ پرداخت ساعتی
- ⏳ محاسبه اضافه‌کاری (1.5 برابر)
- ⚠️ بررسی ورودی‌های نامعتبر
- 🧠 مدیریت خطا با `try/except`
- 📊 نمایش نتیجه با دقت اعشاری

---

## 🛠️ تکنولوژی‌ها

- زبان برنامه‌نویسی: **Python**
- بدون نیاز به کتابخانه خارجی

---

## 📌 نحوه کار برنامه

1. کاربر وارد می‌کند:
   - ⏰ ساعات کاری (hours)
   - 💵 نرخ ساعتی (rate)

2. برنامه بررسی می‌کند:
   - ساعات بین 0 تا 168 باشد
   - نرخ منفی نباشد

3. سپس محاسبه انجام می‌شود:
   - تا 40 ساعت → حقوق عادی
   - بیشتر از 40 ساعت → اضافه‌کاری با ضریب 1.5

4. نتیجه نمایش داده می‌شود
