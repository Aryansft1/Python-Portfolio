# 🔢 مدیریت لیست اعداد (Python)

---

## ✨ توضیح پروژه

این پروژه یک برنامه تعاملی به زبان **Python** است که اعداد وارد شده توسط کاربر را مدیریت می‌کند و بر اساس ورودی‌های خاص، عملیات مختلفی روی لیست انجام می‌دهد.

---

## 🚀 ویژگی‌ها

- ➕ ذخیره اعداد در لیست
- 🔁 مدیریت ورودی‌ها به صورت حلقه بی‌نهایت
- 📊 محاسبه میانگین اعداد مثبت
- 🔄 حذف مقادیر تکراری (اعداد یکتا)
- 🧹 پاک کردن کامل لیست
- ⚠️ مدیریت تعداد زیاد ورودی
- 🧠 استفاده از منطق شرطی پیشرفته
