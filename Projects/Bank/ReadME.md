# 🏦 سیستم مدیریت بانک ساده (Python)

---

## ✨ توضیح پروژه

این پروژه یک سیستم ساده مدیریت بانک به زبان **Python** است که به کاربر اجازه می‌دهد عملیات‌هایی مثل ساخت حساب، واریز، برداشت و مشاهده موجودی را انجام دهد.

این برنامه به صورت **کنسولی (Console Application)** طراحی شده و برای تمرین منطق برنامه‌نویسی و کار با دیکشنری‌ها بسیار مناسب است.

---

## 🚀 ویژگی‌ها

- 👤 ایجاد حساب بانکی جدید
- 💰 واریز پول به حساب
- 💸 برداشت پول از حساب
- 📊 مشاهده موجودی حساب
- ⚠️ بررسی وجود حساب
- 🔁 منوی تکرارشونده (Loop Menu)
- 🧠 مدیریت داده‌ها با `dictionary`

---

## 🛠️ تکنولوژی‌ها

- زبان برنامه‌نویسی: **Python**
- ساختار داده: `dict`
- بدون کتابخانه خارجی

---

## 📌 نحوه کار برنامه

1. کاربر وارد منوی اصلی می‌شود
2. یکی از گزینه‌ها را انتخاب می‌کند:
   - ایجاد حساب
   - واریز
   - برداشت
   - مشاهده موجودی
   - خروج
3. برنامه اطلاعات را پردازش کرده و نتیجه را نمایش می‌دهد
4. تا زمانی که کاربر خارج نشود، برنامه اجرا می‌ماند

---

## 🧠 منطق پروژه

- ✔ استفاده از `dictionary` برای ذخیره حساب‌ها
- ✔ استفاده از حلقه `while True`
- ✔ بررسی شرط وجود حساب
- ✔ مدیریت عملیات مالی ساده
- ✔ جلوگیری از برداشت بیشتر از موجودی
