# 🐍 مجموعه پروژه‌های مقدماتی پایتون (Python Beginner Portfolio)

---

## 👨‍💻 درباره این ریپوزیتوری

این ریپوزیتوری شامل مجموعه‌ای از **پروژه‌های تمرینی سطح مقدماتی پایتون** است که با هدف یادگیری مفاهیم پایه برنامه‌نویسی، منطق الگوریتمی و طراحی برنامه‌های کنسولی ساخته شده‌اند.

---

## 📚 پروژه‌های موجود

### 🏦 1. سیستم مدیریت بانک (Bank System)
- ایجاد حساب کاربری
- واریز و برداشت وجه
- مشاهده موجودی حساب
- مدیریت داده‌ها با `dictionary`
- طراحی منوی تعاملی (Menu System)

---

### 🧮 2. ماشین حساب ساده (Calculator)
- انجام عملیات + ، - ، * ، /
- دریافت ورودی از کاربر
- مدیریت خطا و عملگر نامعتبر
- محاسبات عددی ساده

---

### 🎬 3. سیستم قیمت‌گذاری سینما (Cinema Ticket System)
- محاسبه قیمت بلیت بر اساس سن
- تفاوت قیمت Normal و 3D
- تغییر قیمت بر اساس سانس
- اعمال تخفیف برای کودکان و سالمندان

---

### 🔢 4. مدیریت لیست اعداد (Number List Manager)
- ذخیره اعداد در لیست
- محاسبه میانگین اعداد مثبت
- حذف داده‌های تکراری
- ریست کردن لیست
- کنترل تعداد ورودی‌ها

---

### 💰 5. محاسبه حقوق و اضافه‌کاری (Salary Calculator)
- محاسبه حقوق بر اساس ساعت کاری
- اضافه‌کاری با ضریب 1.5
- بررسی ورودی معتبر
- مدیریت خطا با try/except

---

## 🎯 اهداف آموزشی

این پروژه‌ها با هدف یادگیری موارد زیر طراحی شده‌اند:

- ✔ مبانی زبان Python
- ✔ شرط‌ها (if / else)
- ✔ حلقه‌ها (while loop)
- ✔ کار با لیست و دیکشنری
- ✔ مدیریت ورودی کاربر
- ✔ طراحی منطق برنامه‌های واقعی
